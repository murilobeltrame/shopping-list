# IaC

This folder is reserved for infrastructure-as-code assets and their local Spec-Driven Development assets.

## Contents

- `.specify/`: IaC-local Speckit templates, scripts, and constitution
- `specs/`: infrastructure feature specifications and plans

Infrastructure code has not been added yet. New platform and deployment features should be specified and planned from this folder so they do not inherit application-layer rules.
